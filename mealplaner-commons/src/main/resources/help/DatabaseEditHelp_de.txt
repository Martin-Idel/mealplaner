Gerichtedatenbank editieren

Der zweite Reiter enthält eine Übersicht über alle eingegebenen Gerichte. Hier können Sie einfach Gericht hinzufügen, editieren oder wieder löschen.

Bitte beachten: Jede Änderung erfordert eine Speicherung. Sonst werden die Änderungen nicht gültig!

Für jedes Gericht müssen bestimmte Aspekte eingegeben werden.
- Die Kochzeit: Dies stellt die ungefähre Kochzeit (obere Schranke, unbeaufsichtigte Kochzeiten) für das Gericht, um Ihnen die Möglichkeit zu geben schnell zu kochen, wenn nötig.
- Das primäre Kochutensil: Manchmal möchten Sie vielleicht einen Auflauf kochen, weil dieser im Voraus geplant werden kann. An einem anderen Tag möchten Sie vielleicht einfach ein schnelles Pfannengericht machen, was allerdings nicht möglich ist, wenn Sie für eine größere Anzahl von Leuten kochen.
- Gang: Die meisten Gerichte werden Hauptspeisen sein, aber manchmal möchten Sie den Planer vielleicht auch nutzen, um mehrere Gerichte auf einmal zu kochen.
- Die Präferenz: Insbesondere wenn Sie Familie haben, haben viele Mitglieder sehr starke Meinungen darüber, was schmeckt und was nicht. Manche Gerichte müssen daher wohl häufiger gekocht werden, wohingegen andere für ganz besondere Anlässe reserviert sind.
- Die Stärkebeilage: Theoretisch könnte diese zwar vom Rezept abgeleitet werden, aber da eine Eingabe von Rezepten nicht nötig ist, wenn keine Einkaufsliste erstellt werden soll, muss die Stärkebeilage trotzdem eingegeben werden. Damit wird beim Vorschlag sichergestellt, dass nicht zu häufig dieselbe Beilage in Folge gekocht wird.
- Kommentarfeld: Hier können Sie eintragen, was Sie sich noch merken wollen, sei es die Website mit dem vollen Rezept oder ein paar weitere Hinweise zum Gericht.

Zusätzlich können Sie auch noch die Zutaten für ein Rezept eingeben. Rezepte gibt es zumeist mit Portionsangaben, jedoch weichen diese bekanntermaßen meistens von der Realität ab. Der Kochplaner bietet hier eine einfache Abhilfe, indem man die Zutatenmengen individuell anpassen kann. Die Zutatenlisten werden dann für die automatische Generierung einer Einkaufsliste benutzt.
