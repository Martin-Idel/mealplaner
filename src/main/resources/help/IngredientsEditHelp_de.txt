Der Zutatenreiter

Der dritte Reiter im Hauptfenster ist die Datenbanksicht auf die Zutatenliste. Hier können Sie einfach Zutaten hinzufügen, löschen oder editieren.

Zu beachten: Jede Änderung muss gespeichert werden, wenn die Änderung beibehalten werden soll.

Jede Zutat besteht aus drei Aspekten:
- Einem Namen
- Einem Typ, der dafür verwendet wird die Einkaufsliste so zu organisieren, dass alle Typen beieinander stehen. So stehen z.B. alle Milchprodukte beisammen, wie im Supermarkt.
- Einem Maß, welches bei der Rezepteingabe angezeigt wird (z.B. Gramm oder Liter).

Zutaten können während der Eingabe von Gerichten erstellt werden, allerdings kann es schneller sein, eine große Liste über diesen Reiter vorher anzulegen.
Ferner können Zutaten auch gelöscht werden, zum Beispiel wenn man feststellt, dass sie schon existieren. Es wäre natürlich ein Problem, wenn die Zutaten in Gebrauch sind, weil dann die Rezepte unvollständig werden. Dazu kann beim Löschen ausgewählt werden, mit welcher Zutat die gelöschte Zutat ersetzt werden soll - notfalls kann man auch einfach nicht ersetzen.
